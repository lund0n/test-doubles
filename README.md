# Test Doubles

An example of using Jest's mocking features in tests.

## Quick Start

```bash
$ git clone https://github.com/lund0n/test-doubles
$ cd test-doubles
$ yarn # or npm install if you're using that instead
$ yarn test # or npm test
```
